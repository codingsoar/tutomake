# TutoMake 사용 설명서

## 1. TutoMake가 무엇인가요?

TutoMake는 화면에서 한 작업을 따라 하기 쉬운 튜토리얼로 만들어 주는 프로그램입니다.

예를 들면 이런 일을 할 수 있습니다.

- 프로그램 사용법 안내 만들기
- 게임 조작 설명 만들기
- 사이트 가입 방법 설명 만들기
- 학교 과제용 화면 설명 자료 만들기

사용자가 직접 클릭하고 입력하면서 따라 하는 튜토리얼도 만들 수 있고, 문서나 웹페이지로도 내보낼 수 있습니다.

## 2. 기본 작업 순서

보통은 아래 순서대로 사용하면 됩니다.

1. 새 튜토리얼을 만듭니다.
2. `Record`로 화면 작업을 녹화합니다.
3. `Editor`에서 단계별 설명과 모양을 다듬습니다.
4. `Play`로 직접 실행해 보며 확인합니다.
5. 원하는 형식으로 `Export` 합니다.

## 3. 처음 화면에서 할 수 있는 일

상단 버튼으로 주요 기능을 사용할 수 있습니다.

- `New`: 새 튜토리얼 만들기
- `Save`: 작업 내용을 `.tutomake` 파일로 저장하기
- `Load`: 저장한 튜토리얼 다시 열기
- `Record`: 화면 녹화 시작하기
- `Play`: 만든 튜토리얼 실행해 보기
- `Export`: 여러 형식으로 내보내기
- `View`: 편집 화면의 속성 영역 보이기/숨기기
- `Settings`: 설정과 단축키 확인하기
- 테마 버튼: 밝은 화면과 어두운 화면 바꾸기

## 4. 녹화하기

### 4.1 녹화 시작

`Record` 버튼을 누르면 화면 녹화 준비가 시작됩니다.

- 프로그램 창은 최소화됩니다.
- 전체 화면 기준으로 녹화합니다.
- 잠깐의 카운트다운 뒤에 녹화가 시작됩니다.

### 4.2 자동으로 기록되는 것

녹화 중에는 다음 내용이 자동으로 단계로 저장됩니다.

- 마우스 클릭
- 키보드 입력
- 특수 키 입력
  예: `Enter`, `Tab`, `Esc`, `Space`
- 조합 키 입력
  예: `Ctrl + Z`, `Ctrl + Space`
- 마우스 드래그

### 4.3 녹화가 끝난 뒤

녹화가 끝나면 단계별 화면과 입력 정보가 편집기에 들어옵니다.

이제 각 단계를 수정하면서 더 보기 좋게 만들 수 있습니다.

## 5. 편집하기

편집기는 튜토리얼을 다듬는 곳입니다.

### 5.1 단계 목록

왼쪽 또는 목록 영역에서 각 단계를 고를 수 있습니다.

- 원하는 단계를 선택할 수 있습니다.
- 단계 순서를 바꿀 수 있습니다.
- 필요 없는 단계는 지울 수 있습니다.
- 비슷한 단계는 복제할 수 있습니다.

### 5.2 미리보기 화면

선택한 단계가 화면 위에 어떻게 보일지 확인하는 곳입니다.

- 클릭 영역 위치를 바꿀 수 있습니다.
- 크기를 늘리거나 줄일 수 있습니다.
- 사각형 또는 원 모양으로 바꿀 수 있습니다.

### 5.3 설명 문구

각 단계에는 두 가지 설명을 넣을 수 있습니다.

- `Description`: 짧은 안내 문구
- `Instruction`: 조금 더 자세한 설명

예를 들면 이렇게 나눠 쓸 수 있습니다.

- Description: `로그인 버튼을 누르세요`
- Instruction: `오른쪽 위에 있는 로그인 버튼을 한 번 클릭하세요.`

### 5.4 키보드 단계

키보드 단계는 두 가지 방식으로 만들 수 있습니다.

- `text`: 글자를 직접 입력하는 단계
- `key`: 특수 키나 조합 키를 누르는 단계

예시:

- `hello` 입력
- `Enter` 누르기
- `Ctrl + Z` 누르기

### 5.5 단계 꾸미기

단계 모양도 바꿀 수 있습니다.

- 글자 크기
- 글자 색
- 배경 색
- 테두리 색
- 테두리 두께
- 테두리 선 모양
- 내부 채우기 색
- 내부 투명도

### 5.6 실행 취소와 다시 실행

편집 중 실수해도 괜찮습니다.

- `Undo`: 이전 상태로 되돌리기
- `Redo`: 다시 앞으로 가기

## 6. 재생하기

`Play`를 누르면 사용자가 실제로 따라 하는 튜토리얼처럼 실행됩니다.

- 현재 단계에서 무엇을 해야 하는지 보여 줍니다.
- 클릭 단계에서는 정해진 위치를 눌러야 다음으로 넘어갑니다.
- 키보드 단계에서는 맞는 글자나 키를 입력해야 합니다.
- 드래그 단계에서는 정해진 방향으로 끌어야 합니다.

즉, 그냥 보기만 하는 설명이 아니라 직접 따라 하는 연습형 튜토리얼을 만들 수 있습니다.

## 7. 오디오 기능

오디오 기능도 사용할 수 있습니다.

- 입력 장치를 선택할 수 있습니다.
- 녹음된 오디오를 연결할 수 있습니다.
- 오디오 시작 시점을 앞뒤로 조금 조절할 수 있습니다.

이 기능은 화면 설명에 목소리를 넣고 싶을 때 유용합니다.

## 8. 내보내기

`Export` 메뉴에서 여러 형식으로 저장할 수 있습니다.

### 8.1 동영상 형식

- `MP4`
- `WebM`
- `AVI`
- `GIF`

설명 영상을 만들고 싶을 때 사용합니다.

### 8.2 문서 형식

- `PDF`
- `PowerPoint`
- `Markdown`
- `PNG Sequence`

문서나 발표 자료가 필요할 때 좋습니다.

### 8.3 웹 형식

- `HTML`
- `HTML + Video`
- `Web Embed (JS)`
- `Lottie`

웹페이지에 올리거나 공유하고 싶을 때 사용합니다.

### 8.4 패키지 형식

- `SCORM Package`
- `Portable Package`

학습 시스템이나 묶음 파일로 전달하고 싶을 때 사용합니다.

## 9. 저장 파일

작업 중 저장하는 기본 파일 형식은 `.tutomake` 입니다.

이 파일에는 보통 아래 정보가 들어갑니다.

- 튜토리얼 제목
- 단계 정보
- 설명 문구
- 키보드 입력 정보
- 오디오 관련 정보

작업을 이어서 하려면 `.tutomake` 파일을 잘 보관하면 됩니다.

## 10. 추천 사용 방법

처음 사용할 때는 이렇게 해 보세요.

1. 짧은 작업 하나만 녹화합니다.
2. 단계 설명을 쉬운 말로 바꿉니다.
3. 중요한 버튼만 크게 표시합니다.
4. `Play`로 직접 해 봅니다.
5. 문제가 없으면 `HTML`이나 `PDF`로 내보냅니다.

처음부터 너무 긴 튜토리얼을 만들기보다, 짧게 만든 뒤 점점 늘리는 것이 훨씬 쉽습니다.

## 11. 잘 만드는 팁

- 한 단계에는 한 행동만 넣는 것이 좋습니다.
- 설명은 짧고 분명하게 쓰는 것이 좋습니다.
- 어려운 말보다 쉬운 말을 쓰는 것이 좋습니다.
- 클릭 위치는 너무 작지 않게 잡는 것이 좋습니다.
- 키보드 단계는 실제 사용자가 눌러야 할 키와 똑같이 맞춰야 합니다.
- 내보내기 전에 꼭 `Play`로 한 번 확인하는 것이 좋습니다.

## 12. 자주 생기는 문제

### 12.1 녹화했는데 단계가 이상해요

- 너무 빠르게 조작했는지 확인하세요.
- 필요 없는 단계는 편집기에서 삭제하세요.
- 설명 문구를 직접 수정하세요.

### 12.2 클릭 위치가 잘 안 맞아요

- 편집기 미리보기에서 위치를 다시 조절하세요.
- 크기를 조금 더 크게 잡아 보세요.

### 12.3 키 입력 단계가 원하는 대로 안 보여요

- 일반 글자 입력인지
- 특수 키 입력인지

이 두 가지를 다시 확인하세요.

예를 들어 `hello`는 글자 입력이고, `Enter`는 특수 키 입력입니다.

### 12.4 파일이 너무 많아 보여요

먼저는 `.tutomake` 파일만 저장해서 작업을 이어 가고, 마지막에 필요한 형식만 내보내는 것을 추천합니다.

## 13. 마지막 정리

TutoMake는 화면 작업을 쉽고 보기 좋게 설명 자료로 바꾸는 도구입니다.

중요한 흐름만 기억하면 됩니다.

1. 녹화하고
2. 편집하고
3. 실행해 보고
4. 내보내면 됩니다.

처음에는 간단한 튜토리얼부터 만들어 보세요. 짧은 작업 하나를 제대로 만드는 것이 긴 작업을 대충 만드는 것보다 훨씬 좋습니다.
